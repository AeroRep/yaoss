# yaoss
Yet Another Operating System Setup

## What is yaoss?
- yaoss is a First time setup program for Aero Desktop system.

## dependecies
- tkinter
- sys = default
- subprocess
- pyinstaller >= 6.3.0

## Build
- pyinstaller (your/path/to/app.py) --clean --onedir --onefile -n yaoss --noconsole
